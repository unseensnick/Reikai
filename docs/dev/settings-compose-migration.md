# Settings Compose migration — superseded

**Superseded by the Mihon rebase (2026-06).**

This handbook covered porting settings sections from the Yōkai legacy Conductor + XML-preference stack to Compose + Voyager (the "two-screen" tap / long-press fallback pattern). That pattern does not exist on the Mihon base: Mihon's settings are already Compose + Voyager throughout, with no Conductor controllers and no legacy fallback.

For how settings screens work on the Mihon base, see [.claude/rules/architecture.md](../../.claude/rules/architecture.md) and [.claude/rules/compose-port.md](../../.claude/rules/compose-port.md). The historical Yōkai-era handbook is preserved in git history on the `design/library-compose` branch.
