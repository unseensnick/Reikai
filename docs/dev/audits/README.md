# Audits

Point-in-time audit reports (parity sweeps, security passes, dead-code hunts). Each is a dated snapshot of findings, not a living document: its action items get filed into [ROADMAP.md](../../../ROADMAP.md) as backlog lines, and the report is kept here only for reference.

**Contents are local (gitignored); this README is the only tracked file.** Audits can name specifics freely (they are a working record, not a public surface). Name them `YYYY-MM-DD-<topic>.md`.
